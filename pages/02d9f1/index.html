# 📚 资料

  • Seata (opens new window) - 分布式事务中间件。
  • ShardingSphere (opens new window) - 关系型数据库读写分离、分库分表中间件。
  • Flyway (opens new window) - 关系型数据库版本管理中间件。
  • Canal (opens new window) - 基于 MySQL 的 binlog，提供增量数据订阅和消费。
  • Twemproxy (opens new window) - Twitter 开源的一个 Redis 和 Memcache 的中间代理服务。
  • Codis (opens new window) - Redis 分布式集群方案。
